Tecnologias usadas neste projeto

HTML

HTML (Hypertext Markup Language) é a linguagem padrão para criar páginas web. Utilizado para estruturar o conteúdo, incluindo textos, imagens, links e outros elementos multimídia.

CSS

CSS (Cascading Style Sheets) é usado para descrever a apresentação de um documento escrito em HTML ou XML. Utilizado para dar estilo ao layout, cores, fontes e outros aspectos visuais do site.

JavaScript

JavaScript é uma linguagem de programação que permite implementar funcionalidades complexas em páginas web, como atualizações de conteúdo dinâmico, animações, controlo de multimídia e muito mais.

Python

Python é uma linguagem de programação de alto nível usada tanto no backend quanto no desenvolvimento web. É conhecido pela sua simplicidade e eficiência, sendo ideal para desenvolvimento de aplicações.

SQL

SQL (Structured Query Language) é uma linguagem padrão para gestão e manipulação de dadoS. Utilizado para executar tarefas como consultas, inserções, atualizações e exclusões de dados.
